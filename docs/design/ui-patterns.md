# UI 패턴

> 반복되는 UI 패턴 정리
> 최종 업데이트: 2026-02-20

## 1. 확인 카드 패턴

채팅에서 생성된 일정을 사용자에게 확인받는 카드.

```
┌─────────────────────────────┐
│ 헤더 (제목 + 일정 수)       │
│ ┌─────────────────────────┐ │
│ │ 시간  제목  기간  뱃지  │ │
│ │ ...                     │ │
│ └─────────────────────────┘ │
│ [전체 등록]  [취소]         │
└─────────────────────────────┘
```

### 변형

| 카드 | 테마 | 컴포넌트 |
|------|------|----------|
| 일정 확인 | blue/orange/red/purple | `ScheduleCard.jsx` |
| 일상 배치 | green | `BatchConfirmCard.jsx` |
| 펫 케어 | teal | `PetCareCard.jsx` |
| 업무 | indigo | `WorkScheduleCard.jsx` |

## 2. 멀티데이 템플릿 패턴

7일/30일치 일정을 생성할 때 1일치 템플릿을 보여주고 기간을 선택.

```
┌─────────────────────────────┐
│ 헤더 (N일 · M개 일정)       │
│ 날짜 범위 표시              │
│ ── 1일치 템플릿 ──          │
│ ┌─────────────────────────┐ │
│ │ 이벤트 목록              │ │
│ └─────────────────────────┘ │
│ [전체 등록]  [취소]         │
└─────────────────────────────┘
```

## 3. 온보딩 플로우 패턴

도우미 최초 사용 시 단계별 질문.

```
채팅 메시지 (질문) → 버튼 선택 → 다음 질문 → ... → GPT 호출 → 카드 표시
```

## 4. 탭 전환 패턴

`chatMode` state로 우측 패널 뷰 전환.

```
chatMode 값    렌더링 컴포넌트
────────────────────────────
'chat'         ChatInterface
'schedule'     DailyScheduleView
'major'        MajorEventsView
'petcare'      DailyScheduleView (petCareMode)
'work'         WorkScheduleView
```

## 5. 개별 항목 제거 패턴

카드 내 각 항목에 [-] 버튼, 클릭 시 해당 항목만 제거.

## 6. 프로필 저장/로드 패턴

```
Firestore: helperProfiles/{userId}_{helperId}
최초 온보딩 → 프로필 저장 → 이후 프로필 로드 → 변경 사항만 재입력
```
