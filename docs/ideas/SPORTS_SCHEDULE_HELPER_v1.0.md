# 스포츠 스케줄 도우미 기획서 v1.0

> 야구, 축구, 농구 등 프로 스포츠 경기 일정을 자동으로 캘린더에 등록하는 도우미
> 작성일: 2026-02-19 | 버전: 1.0

---

## 개요

사용자가 응원하는 팀을 등록하면, 해당 팀의 경기 일정, 포스트시즌, 올스타전 등을 자동으로 캘린더에 추가하는 도우미 기능.

**핵심 가치**
- 내 팀 경기 놓침 방지 — "오늘 경기 몇 시지?" 고민 끝
- 채팅 기반 등록 — "두산 경기 일정 알려줘" 한마디로 자동 등록
- 다중 팀/종목 관리 — 야구, 축구, 농구 여러 팀 일정을 하나의 캘린더에서 통합

## 지원 일정 유형

### 1. 정규시즌 경기
- 내 팀 경기 일정 (날짜, 시간, 상대팀, 장소)
- 홈/원정 구분
- 중계 채널 정보

### 2. 포스트시즌/플레이오프
- 와일드카드, 준플레이오프, 플레이오프, 한국시리즈 (KBO)
- 파이널 라운드, 챔피언결정전 (KBL)
- 플레이오프, FA컵, 리그컵 (K리그)

### 3. 특별 경기/이벤트
- 올스타전
- 개막전/개막식
- 더비 매치 (라이벌전)
- 국가대표 A매치

### 4. 시즌 일정
- 시즌 개막일/폐막일
- 이적 시장 마감일
- 드래프트

### 5. 국제 대회
- 월드컵/월드컵 예선
- WBC (야구)
- 올림픽
- AFC 챔피언스리그

## 사용자 플로우

### 최초 온보딩

```
1. 채팅에서 스포츠 관련 요청
   ├─ "두산 경기 일정 알려줘"
   ├─ "이번 주 KBO 일정"
   ├─ "프리미어리그 일정 등록"
   └─ 또는 도우미 선택 UI에서 "스포츠 스케줄 도우미" 클릭

2. 온보딩 질문 (2~4개)
   ├─ "어떤 종목에 관심 있으세요?" → ⚾ 야구 / ⚽ 축구 / 🏀 농구 (복수 선택)
   ├─ "어떤 팀을 응원하세요?" → 팀 검색/선택 (복수 가능)
   ├─ "어떤 일정을 받으시겠어요?" → 전체 경기 / 홈 경기만 / 주말 경기만
   └─ "경기 몇 시간 전에 알려드릴까요?" → 당일 / 1시간 전 / 3시간 전

3. AI가 경기 일정 생성 (카드 목록)
   ├─ 03/29 (토) 14:00  ⚾ 두산 vs LG (잠실) - 개막전
   ├─ 03/30 (일) 14:00  ⚾ 두산 vs LG (잠실)
   ├─ 04/01 (화) 18:30  ⚾ 삼성 vs 두산 (대구)
   ├─ 04/02 (수) 18:30  ⚾ 삼성 vs 두산 (대구)
   └─ ...

4. 사용자 확인/수정
   ├─ "전체 등록" → 일괄 Firestore 저장
   ├─ "홈 경기만 등록" → 필터 후 등록
   ├─ "이번 달만 등록" → 기간 필터
   └─ "다시 알려줘" → 재검색
```

### 후속 사용

```
1. 팀/종목 추가
   └─ "토트넘 경기도 추가해줘" → EPL 팀 추가

2. 일정 갱신
   └─ "이번 달 경기 일정 업데이트해줘" → 최신 일정으로 갱신

3. 특정 조회
   ├─ "이번 주 두산 경기"
   ├─ "다음 홈 경기 언제야?"
   └─ "이번 달 주말 경기만"

4. 팀 삭제
   └─ "NBA 일정 더 이상 안 받을래"

5. 결과 확인
   └─ "어제 두산 경기 결과" → 스코어 안내 (Phase 2)
```

## 데이터 소스 전략

### MVP: GPT 지식 기반 생성

GPT-4o-mini의 학습 데이터를 활용하여 시즌 일정의 대략적 패턴을 생성한다.

**한계점**: 정확한 경기 날짜/시간은 매 시즌 새로 편성되므로 GPT 단독으로는 최신 일정 제공이 어려움

**보완 전략**:
- 시즌 개막일, 올스타전 등 확정 일정은 정확히 제공
- 정규시즌 개별 경기는 `(추정)` 라벨 표시
- 사용자에게 "공식 일정 확정 후 갱신해드릴까요?" 안내

### Phase 2: 외부 API 연동

| 소스 | 대상 | 데이터 |
|------|------|--------|
| KBO 공식 API/크롤링 | KBO 전 경기 | 정확한 경기 일정, 결과 |
| K리그 공식 API | K리그1/2 | 경기 일정, 순위 |
| KBL 공식 사이트 | KBL 전 경기 | 경기 일정, 결과 |
| ESPN API | MLB/NBA/EPL 등 | 해외 리그 일정, 결과 |
| Football-Data.org | 유럽 축구 | 경기 일정, 스탯 |

### Phase 3: 실시간 정보

- 경기 시작 전 알림
- 실시간 스코어 업데이트
- 경기 결과 자동 기록
- 일정 변경(우천 취소 등) 자동 감지

### 스포츠 이벤트 스키마

```javascript
// 기존 events 컬렉션에 저장
{
  userId: 'uid',
  title: '⚾ 두산 vs LG (잠실)',
  startTime: Timestamp,          // 경기 시작 시간
  endTime: Timestamp,            // 예상 종료 시간
  category: '스포츠',
  location: '잠실야구장',
  attendees: [],
  createdAt: Timestamp,
  createdVia: 'helper',

  // 스포츠 도우미 전용 메타데이터
  helperMeta: {
    helperId: 'H12',
    teamId: 'kbo_doosan',
    teamName: '두산',
    sport: 'baseball',
    league: 'KBO',
    matchType: 'regular',       // 경기 유형 (아래 표 참조)
    opponent: 'LG 트윈스',
    isHome: true,               // 홈 경기 여부
    venue: '잠실야구장',
    broadcast: 'KBS N',         // 중계 채널 (알 수 있는 경우)
    isEstimated: false
  }
}
```

### 경기 유형 분류

| matchType | 한글 | 카드 색상 | 비고 |
|-----------|------|----------|------|
| `regular` | 정규시즌 | 파란색 (blue) | 기본 경기 |
| `postseason` | 포스트시즌 | 빨간색 (red) | 플레이오프, 한국시리즈 등 |
| `allstar` | 올스타전 | 보라색 (purple) | 특별 경기 |
| `cup` | 컵 대회 | 노란색 (yellow) | FA컵, 리그컵 등 |
| `international` | 국제 대회 | 초록색 (green) | 월드컵, WBC, UCL 등 |
| `derby` | 더비/라이벌전 | 주황색 (orange) | 잠실 더비, 엘클라시코 등 |
| `opening` | 개막전 | 금색 (amber) | 시즌 첫 경기 |

## 기술 의사결정

### GPT 기반 vs 외부 API 기반

| 방식 | 장점 | 단점 |
|------|------|------|
| **GPT 기반 (MVP 채택)** | 구현 간단, 모든 리그 대응 | 개별 경기 날짜 부정확 가능 |
| **외부 API 기반** | 정확한 실시간 일정 | 리그별 API 다름, 유지보수 비용 |
| **하이브리드 (Phase 2 목표)** | GPT 기본 + 국내 리그 API 보정 | 구현 복잡도 증가 |

**MVP 결정**: GPT 기반 — 시즌 일정 패턴(개막일, 요일별 경기 시간 등)을 GPT가 생성하고, `isEstimated` 플래그로 정확도 투명성 확보. 정확한 상대팀 매칭이 어려운 경우 "두산 홈 경기"처럼 일반화하여 표시

### 데이터 범위

| 범위 | 채택 | 이유 |
|------|------|------|
| 국내 리그 (KBO/K리그/KBL) | **MVP** | 한국 사용자 타겟, 데이터 정확도 높음 |
| 주요 해외 리그 (EPL/MLB/NBA) | Phase 2 | 팬층 존재하나 API 연동 필요 |
| 전체 글로벌 | Phase 3+ | 사용자 요청 기반으로 점진 확장 |
