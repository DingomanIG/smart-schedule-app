# 육아 스케줄 도우미 아이디어

> ID: H06 | 우선순위: Phase 2

## 한줄 요약

아이의 월령에 맞춰 수유, 낮잠, 놀이, 목욕, 예방접종 등 육아 스케줄을 자동 생성하는 도우미.

## 핵심 가치

- "8개월 아기" 한마디로 이유식/낮잠/놀이 루틴 완성
- 다자녀 스케줄 통합 관리
- 출생일 기준 국가 필수 예방접종 일정 자동 계산

## 월령별 스케줄 구간

| 구간 | 월령 | 핵심 변화 |
|------|------|----------|
| 신생아 | 0~2개월 | 2~3시간 간격 수유, 16~18시간 수면 |
| 초기 영아 | 3~5개월 | 3~4시간 수유, 낮잠 3회 |
| 이유식 시작 | 6~8개월 | 이유식 1~2회 + 수유, 낮잠 2회 |
| 이유식 중기 | 9~11개월 | 이유식 3회 + 수유, 낮잠 2회 |
| 돌 전후 | 12~17개월 | 유아식 전환, 낮잠 1~2회 |
| 걸음마기 | 18~24개월 | 유아식 3끼, 낮잠 1회, 배변훈련 |
| 유아기 | 25~36개월 | 성인 유사 식사, 낮잠 0~1회 |

## 카테고리

수유/식사(amber), 수면(indigo), 놀이(green), 목욕(sky), 기저귀(orange), 외출(emerald), 병원(red), 발달 체크(violet)

## 사용 시나리오

1. 아이 등록 (이름, 생년월일, 성별) → 월령 자동 계산
2. 수유/수면/활동 설정 → 월령 구간 템플릿 기반 스케줄 자동 생성
3. 규칙 기반 생성 (GPT 불필요), 기간(1일/7일/30일) 선택 후 등록
4. 예방접종 일정 자동 생성 + 완료 체크
5. 다자녀 시 시간순 병합 표시

## 향후 확장

- 성장 기록 (키/몸무게 그래프, WHO 표준 비교)
- 발달 체크리스트 (뒤집기, 앉기, 걷기 등)
- 수유/수면 기록 + 패턴 분석
- 월령 구간 변경 시 자동 전환 알림
- 어린이집 등/하원 시간 연동
