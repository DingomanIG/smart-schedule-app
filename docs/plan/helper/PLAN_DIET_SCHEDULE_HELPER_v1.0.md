# 다이어트 스케줄 도우미 아이디어

> ID: H05 | 우선순위: Phase 2

## 한줄 요약

다이어트 목표(감량/유지/증량)에 맞춰 식단과 운동을 통합 관리하는 스케줄 도우미.

## 핵심 가치

- 식단 + 운동을 하나의 플로우로 통합 관리
- 목표 체중 기반 칼로리 자동 계산 (BMR/TDEE)
- 주간 단위 식단/운동 스케줄 일괄 생성

## 주요 기능

- **신체 정보 입력**: 성별, 나이, 키, 현재/목표 체중
- **자동 칼로리 계산**: Mifflin-St Jeor 공식 기반 BMR → TDEE → 목표 칼로리
- **식단 배분**: 식사 횟수(3~5끼)별 칼로리 배분, 영양소 비율(탄/단/지)
- **운동 설정**: 수준(입문~고급), 주 횟수, 선호 운동, 요일별 루틴 분배
- **수분 섭취 알림**: 목표량, 간격별 알림

## 카테고리

아침(amber), 점심(orange), 저녁(red), 간식(pink), 유산소(sky), 근력(indigo), 스트레칭(violet), 수분(cyan)

## 사용 시나리오

1. 프로필 설정 (신체, 목표, 식단, 운동, 수분)
2. 규칙 기반으로 일일 스케줄 자동 생성 (GPT 호출 불필요)
3. 기간 선택 (1일/7일/30일) 후 캘린더에 배치 등록
4. 식단 추천 시 GPT 호출하여 구체적 한식 메뉴 제안

## 향후 확장

- GPT 기반 구체적 메뉴 추천
- 체중 기록 + 그래프 시각화
- 칼로리 실적 추적 (완료 체크)
- 사진 기반 식단 기록 (AI 칼로리 추정)
