# 게임 스케줄 도우미 아이디어

> ID: H10 | 우선순위: Phase 2

## 한줄 요약

플레이하는 게임을 등록하면 업데이트, 시즌, 이벤트 일정을 자동으로 캘린더에 추가하는 도우미.

## 핵심 가치

- 시즌 시작, 한정 이벤트, 패치 등을 놓치지 않도록 관리
- "발로란트 일정 알려줘" 한마디로 게임 일정 자동 등록
- 여러 게임의 일정을 하나의 캘린더에서 통합 관리

## 지원 일정 유형

- 시즌/에피소드 출시 (배틀패스, 랭크 리셋 등)
- 패치/업데이트 (정기 점검, 대규모 패치)
- 한정 이벤트 (콜라보, 무료 보상)
- 신작/DLC 출시 (얼리 액세스, 사전예약)
- e스포츠 대회 (결승전, 시청 보상 드롭)

## 대상 게임 (예시)

발로란트, LoL, 겐신 임팩트, 포트나이트, 오버워치 2, 메이플스토리, 에이펙스, 스타레일 등

## 데이터 소스 전략

- MVP: GPT 지식 기반 생성 (확정/추정 구분, `(추정)` 라벨)
- Phase 2: 웹 크롤링, Steam 뉴스 API, RSS 연동
- Phase 3: 커뮤니티 기반 일정 DB

## 사용 시나리오

1. 게임 선택 (복수 가능) → 알림 유형 선택 → AI가 향후 1~2개월 일정 생성
2. 카드 목록으로 확인 후 일괄 등록
3. 이후 게임 추가/제거, 일정 갱신 가능

## 향후 확장

- 웹 검색 연동으로 정확도 향상
- 일정 자동 갱신 (주 1회)
- 충돌 감지 ("패치 시간에 회의가 있어요")
- 플레이타임 기반 추천 우선순위
