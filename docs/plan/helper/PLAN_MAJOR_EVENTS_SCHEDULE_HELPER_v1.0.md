# 주요일정 스케줄 도우미 아이디어

> ID: H12 | 우선순위: Phase 2

## 한줄 요약

공휴일, 생일, 기념일, 시즌 이벤트 등 반복되는 주요 일정을 매년 자동 등록하고 사전 알림하는 도우미.

## 핵심 가치

- 공휴일/대체공휴일 자동 반영 (한국 맞춤)
- 생일(음력 포함), 기념일(100일 단위), 기일 등 개인 일정 관리
- 연차 효율 분석 ("1일 연차로 4일 연휴" 추천)

## 일정 카테고리

1. **공휴일**: 대한민국 공휴일 + 대체공휴일 자동 등록
2. **연휴 분석**: 황금연휴, 징검다리 연휴 감지, 연차 최적화 추천
3. **생일**: 양력/음력 지원, 관계 태그, D-Day, 선물 메모
4. **기념일**: 연인(100일 단위 자동계산), 결혼, 입사일, 추모일
5. **시즌 이벤트**: 벚꽃, 장마, 김장, 연말정산, 발렌타인 등
6. **24절기** (선택): 매년 변동 날짜 자동 계산

## 사용 시나리오

- "엄마 생일 등록해줘. 음력 3월 15일이야" → 음력→양력 변환 후 매년 자동 등록
- "올해 연차 효율 좋은 날 알려줘" → TOP 5 추천
- "여자친구랑 사귄 날 3월 14일이야" → 100일, 200일, 1주년 자동 생성

## 관계별 알림 전략

- 가족: 1주전, 3일전, 당일
- 연인: 1주전, 3일전, 1일전, 당일
- 친구: 3일전, 당일
- 직장동료: 1일전, 당일

## 향후 확장

- 가족/커플 간 기념일 공유
- GPT 기반 선물 추천, 축하 메시지 생성
- 해외 공휴일 지원 (글로벌 확장)
- D-Day 위젯
