# 펫 케어 스케줄 도우미 아이디어

> ID: H11 | 우선순위: Phase 2

## 한줄 요약

반려동물 종류와 정보를 입력하면 최적의 일일 돌봄 스케줄을 자동 생성하는 도우미.

## 핵심 가치

- 밥, 산책, 화장실 청소 등 돌봄 루틴 체계화
- 강아지/고양이 각각 다른 돌봄 패턴 반영
- 아기/성체/노령 단계별 케어 주기 최적화

## 지원 동물 및 돌봄 항목

**강아지**: 밥(2회), 산책(2회), 배변 패드, 놀이, 양치, 빗질, 심장사상충 예방, 정기검진
**고양이**: 밥(2회), 물 교체, 화장실 청소(1~2회), 놀이, 빗질, 구충제, 정기검진

## 성장 단계별 차이

- 아기(0~6개월): 밥 3~4회, 짧은 놀이, 매월 접종
- 성체: 표준 케어 (밥 2회, 적정 운동)
- 노령: 소량 식사, 부드러운 놀이, 잦은 건강 체크

## 사용 시나리오

1. 동물 종류, 이름, 나이, 크기 입력 → 돌봄 항목 선택
2. AI가 보호자 생활 패턴에 맞춘 하루 스케줄 생성
3. 유형별 색상 카드로 확인 후 일괄 등록
4. 다중 반려동물 통합 스케줄 지원

## 향후 확장

- 반복 일정 등록 (매일/주간/월간)
- 완료 체크 기능
- 성장에 따른 스케줄 자동 조정
- 날씨 연동 (비오는 날 산책 조정)
- 햄스터, 토끼, 앵무새 등 확장 동물 지원
