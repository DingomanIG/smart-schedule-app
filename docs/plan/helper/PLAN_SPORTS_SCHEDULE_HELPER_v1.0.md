# 스포츠 스케줄 도우미 아이디어

> ID: H12 | 우선순위: Phase 2

## 한줄 요약

응원하는 팀을 등록하면 경기 일정, 포스트시즌, 올스타전 등을 캘린더에 자동 추가하는 도우미.

## 핵심 가치

- "오늘 경기 몇 시지?" 고민 없이 내 팀 경기 자동 관리
- "두산 경기 일정 알려줘" 한마디로 등록
- 야구, 축구, 농구 여러 팀 일정을 하나의 캘린더에서 통합

## 지원 종목 및 리그

- **야구**: KBO (10팀), MLB, NPB
- **축구**: K리그1 (12팀), EPL, 라리가, 분데스리가, 세리에A, 국가대표
- **농구**: KBL (10팀), WKBL, NBA

## 지원 일정 유형

- 정규시즌 (홈/원정, 상대팀, 장소, 중계)
- 포스트시즌/플레이오프
- 올스타전, 개막전, 더비 매치
- 국제 대회 (월드컵, WBC, AFC CL)

## 데이터 소스 전략

- MVP: GPT 기반 시즌 패턴 생성 (확정/추정 구분)
- Phase 2: KBO/K리그/KBL 공식 API, ESPN API 연동
- Phase 3: 실시간 스코어, 경기 결과 자동 기록

## 사용 시나리오

1. 종목/팀 선택 → 전체/홈/주말 경기 필터 → 알림 설정
2. AI가 향후 1개월 경기 일정 카드 목록 생성
3. 홈/원정 구분, 라이벌전 강조 표시
4. 일괄 등록 또는 필터 등록

## 향후 확장

- 경기 결과/순위표 조회
- 우천 취소 등 일정 변경 자동 반영
- 직관 모드 (홈 경기 + 주변 경기장 추천)
- 배구, 골프, 테니스, F1, e스포츠 확장
