# 업무 스케줄 도우미 아이디어

> ID: H04 | 우선순위: Phase 2

## 한줄 요약

근무 환경과 태스크 목록을 입력하면 타임블록킹 기반 최적 업무 스케줄을 자동 설계하는 도우미.

## 핵심 가치

- 집중 시간대에 핵심 업무 배치, 저에너지 시간에 정리 업무 배치
- 마감 역산으로 태스크 분배, 긴급도/중요도 기반 우선순위
- 포모도로 기반 집중/휴식 리듬으로 번아웃 방지

## 업무 카테고리

딥워크(indigo), 회의(violet), 정리(gray), 기획(blue), 소통(cyan), 휴식(green), 출퇴근(slate), 마감(red)

## 스케줄링 원칙

- 딥워크 블록: 60~120분 단위, 사이에 15분 휴식
- 고정 회의 전후 15분 버퍼 확보
- 점심 직후 30분은 저강도 업무 배치
- 이메일/메신저 확인은 하루 2~3회로 묶기
- 하루 시작/끝에 계획/마무리 시간 확보

## 사용 시나리오

1. 근무 형태(사무직/재택/하이브리드/프리랜서), 시간, 집중 패턴 설정 (1회)
2. 매일 태스크 목록 + 마감일 입력
3. AI가 최적 타임블록 스케줄 생성 → 카드로 확인 후 등록
4. 주간 모드: 월~금 5일치 일괄 생성

## 향후 확장

- 기존 캘린더 이벤트와 충돌 감지 + 자동 조정
- 포모도로 세분화 옵션
- 태스크 완료 추적 + 일일 달성률
- 패턴 학습 (매주 월요일 팀미팅 자동 감지)
- 팀 캘린더 연동 (회의 최적 시간 제안)
